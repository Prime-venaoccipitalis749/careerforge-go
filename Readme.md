## 🧭 What this app does

CareerForge Go helps you upload a resume, review the file, and view a simple analysis on your Windows PC. It is built for people who want a quick way to check a resume without setting up extra tools.

The app includes a desktop build for Windows and an Android app build. For most Windows users, the main path is to download the release, open the file, and start using it.

## 💻 What you need

- A Windows 10 or Windows 11 PC
- Internet access for the first download
- Enough free space for the app and your resume files
- A mouse or trackpad
- A PDF, DOCX, or similar resume file

## ⬇️ Download the app

## 🪟 Install on Windows

1. Open the release page.
2. Find the latest release at the top.
3. Download the Windows file.
4. If the file is zipped, right-click it and choose Extract All.
5. Open the extracted folder.
6. Double-click the `.exe` file to start the app.
7. If Windows asks for permission, choose Run.

If you see more than one file, pick the one that looks like the desktop app for Windows. The file name may include `setup`, `windows`, `exe`, or `desktop`.

## ▶️ Run the app

After the app opens:

1. Sign in if the app asks for it.
2. Open the profile or resume upload screen.
3. Choose a resume file from your computer.
4. Upload the file.
5. Wait for the analysis to finish.
6. Review the results on screen.

The app is set up to show resume feedback in a clear format. You can use it to check profile content, resume structure, and basic matching details.

## 📁 How to upload a resume

1. Open the upload section in the app.
2. Click the upload button.
3. Select your resume file.
4. Confirm the file choice.
5. Wait for the upload to finish.

For best results, use a clean file name like:

- `John_Smith_Resume.pdf`
- `Resume_2026.docx`
- `Career_Profile.pdf`

Keep the file in a place you can find fast, such as Downloads or Desktop.

Use names like these:

- `resume.pdf`
- `careerforge_profile.docx`
